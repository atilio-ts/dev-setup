## Restoring on a new machine

```bash
git clone git@github.com:youruser/dev-setup.git
cd dev-setup
bash setup.sh
```

The script handles: Homebrew, all packages, Oh My Zsh, shell config, git, Neovim, atuin, gh, VS Code settings, Claude Code (config + hooks + agents + rules + global memory), spicetify, navi cheatsheets, and macOS Dock settings.

> **Note:** Files marked *snapshot* in `claude/` are auto-generated by the ECC plugin and tracked here for reference only — `setup.sh` does not copy them (the plugin installer recreates them on a new machine).

See DEV_SETUP.md for anything that requires manual steps (Docker, JetBrains, Java JDKs, Claude skills).
